# AI Resume Parser & ATS Analyzer

An AI-powered Resume Parser and ATS Score Analyzer built using MERN Stack and FastAPI.

## Features

- Resume Upload (PDF)
- AI Resume Parsing
- ATS Score Calculation
- Skill Extraction
- Resume Dashboard
- AI Suggestions
- MongoDB Integration
- FastAPI NLP Parser

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### AI Parser
- FastAPI
- Python
- spaCy
